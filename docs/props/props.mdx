# Props

## Component

<Props of={ReactGlobe} />

## `CameraOptionsProp`

Use the `cameraOptions` prop to control the globe camera (i.e. how the globe is viewed). You can pass any number of option fields. Default options can be imported from the module with:

```js
import { defaultCameraOptions } from 'react-globe';
```

<Props of={CameraOptionsProp} />

## `FocusOptionsProp`

Use the `focusOptions` prop to control how focused coordinates behave. You can pass any number of option fields. Default options can be imported from the module with:

```js
import { defaultFocusOptions } from 'react-globe';
```

<Props of={FocusOptionsProp} />

## `GlobeOptionsProp`

Use the `globeOptions` prop to control the looks and behaviors of the globe. You can pass any number of option fields. Default options can be imported from the module with:

```js
import { defaultGlobeOptions } from 'react-globe';
```

<Props of={GlobeOptionsProp} />

## `LightOptionsProp`

Use the `lightOptions` prop to control the color and intensity values of ambient and point lights. The ambient light illuminates all objects in the scene equally while the point light provides a source of light emitting from a single point. You can pass any number of option fields. Default options can be imported from the module with:

```js
import { defaultLightOptions } from 'react-globe';
```

<Props of={LightOptionsProp} />

## `MarkerOptionsProp`

Use the `markerOptions` prop to control the looks and behavior of markers. If you provide a valid marker `type` (e.g. `bar`, `dot`), `react-globe` will render the corresponding supported marker. You can pass any number of option fields. Default options can be imported from the module with:

```js
import { defaultMarkerOptions } from 'react-globe';
```

<Props of={MarkerOptionsProp} />

## `Animation`

An `Animation` is a Javascript object with four required fields that describe how an animation should behave. `react-globe` will sequentially execute the animation steps in an array of animations.

<Props of={AnimationsProp} />

## `Marker`

A `Marker` is a Javascript object with required `coordinates` and `size` fields that describe how it should be placed and drawn on the globe. It can contain any other additional fields that are useful for rendering tooltips and handling pointer events.

<Props of={MarkersProp} />
